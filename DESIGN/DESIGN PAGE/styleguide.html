Website Style Guide & Wireframe

This page outlines the visual design system and layout foundation used in my website. It documents type, color, UI elements, and layout structure, supporting a consistent and user-friendly experience.

Typography

Font Family: Inter

H1 (2.5rem / 800 weight): Used for main page titles to grab user attention with strong visual hierarchy.

H2 (2rem / 700 weight): Used for section headers to clearly separate content areas.

Body Text (1rem / 400 weight): Designed for readability in paragraphs, blogs, and general content.

The Inter typeface was chosen for its modern, clean look and strong legibility across devices and screen sizes.

Color Palette

#2c3e50 (Text): A dark blue-grey used for body text and headings to provide strong contrast and readability.

#f7e7ce (Background): A soft beige that provides a calm, warm base for content areas without overwhelming the eyes.

#58738e (Accent): A muted navy-blue tone used for buttons, links, and interactive elements to attract user attention.

#d8a7b1 (Highlight): A soft rose-pink used to emphasize certain sections or active elements for visual interest.

This palette was selected to maintain visual harmony, accessibility, and a soft, creative feel aligned with the brand tone.

Buttons

Primary buttons are styled with the accent color (#58738e) and white text for contrast. They are used for key interactions like submitting forms or navigating to important sections. Hover and active states provide feedback to ensure intuitive interaction.

Sample Wireframe

Home Page Layout

  • Header: Contains site logo and navigation bar for intuitive access to all site sections.
  • Hero Image: Features a large image/banner with a welcome message and call-to-action button to engage users.
  • Blog Preview Grid: Showcases recent posts in a clean, card-style layout to improve content discoverability.
  • Footer: Includes links to social media, contact info, and other utility links for better site navigation.

This wireframe was created to reflect a mobile-first, responsive layout, following design principles like hierarchy, spacing, and balance.
