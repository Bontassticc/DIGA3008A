WEEK 12: BLOG POST

Reflection on how my coding intersects with decolonialism

Reflecting on how my coding practices intersect with decolonialism, I recognize that the principles of accessibility, inclusivity, and cultural sensitivity are crucial. In the context of web development, this means being mindful of design choices, content presentation, and technical structures that respect diverse user experiences and avoid perpetuating digital inequalities.

Decolonial Considerations in My Coding Practice:

  • Inclusivity Design and Accessibility: Ensuring that my website is accessible to users with varying abilities by implementing semantic HTML, ARIA roles, keyboard navigation, and text-to-speech support.
  • Language Sensitivity:Avoiding culturally insensitive language, stereotypes, or imagery that may alienate users from different backgrounds.
  • User-Centric Design:Prioritizing user needs, particularly for those who may face digital barriers (such as users in low-bandwidth regions).
  • Ethical Data Practices: Avoiding unnecessary data collection and ensuring that any analytics do not infringe on user privacy.

JavaScript Integration in My Website

To enhance my website, i have intergrated JavaScript for various interactive elements. The main changes I made may include:

  • Interactive Navigation:JavaScript enhances the user experience with dynamic navigation menu and smooth scrolling, making the site more user-friendly.
  • Form Validation:Implemented client-side form validation to guide users without needing a page reload.
  • Content Filtering and Sorting: Added interactive blog filtering, allowing user to easily access content that interests them.
  • Responsive Elements: JavaScript ensures certain components, like dropdowns and modal popups, are accessible across different screen sizes.

Impact on Interaction Design:

These JavaScript enhancements streamline interaction design by:

  • Making the user experience more dynamic and responsive
  • Reducing page load times with client-side interactions
  • Allowing for personalised interaction, such as tailored content display.
  • Enhancing accessibility by providing instant feedback (like form validation)
